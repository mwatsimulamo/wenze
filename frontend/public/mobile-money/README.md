# Logos Mobile Money

Ce dossier contient les logos des réseaux Mobile Money supportés.

## Fichiers requis

Placez les fichiers PNG suivants dans ce dossier :

1. **mpesa.png** - Logo M-Pesa
2. **airtel.png** - Logo Airtel Money
3. **orange.png** - Logo Orange Money
4. **africell.png** - Logo Africell Money

## Spécifications recommandées

- **Format** : PNG avec fond transparent
- **Taille** : 256x256 pixels minimum (ou plus grand, sera redimensionné automatiquement)
- **Qualité** : Haute résolution pour un rendu net

## Où trouver les logos

Vous pouvez télécharger les logos officiels depuis :
- Sites officiels des opérateurs
- Repositories de logos (ex: Simple Icons, Brandfetch)
- Ressources de design officielles

## Note

Si un logo n'est pas trouvé, il sera automatiquement masqué dans l'interface.
